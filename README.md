# 麻雀点数管理アプリ (Mahjong Score Manager)

ブラウザで動作する、インストール不要の麻雀点数管理＆計算ツールです。
4人打ち・3人打ちに対応し、複雑な点数計算も内蔵されたツールで行えます。

## 機能
- **点数管理**: リアルタイムで点数を表示・移動
- **点数計算ツール**: 符・翻・役から点数を自動計算（アプリ内に統合）
- **対応ルール**: 4人打ち / 3人打ち（ツモ損あり/なし対応）
- **UI**: スマホ・タブレット・PC対応のレスポンシブデザイン

## 使い方
1. `index.html` を開くとスタート画面が表示されます。
2. プレイヤー名を入力して「ゲーム開始」を押します。
3. 起家（最初の親）を選択してゲームを始めます。

## GitHub Pagesでの公開方法
このリポジトリをGitHub Pagesで公開すると、Webサイトとして誰でもアクセスできるようになります。

1. GitHubリポジトリの **Settings** (設定) タブを開く
2. 左メニューの **Pages** をクリック
3. **Build and deployment** > **Source** で `Deploy from a branch` を選択
4. **Branch** で `main` (または `master`) を選択し、フォルダは `/ (root)` のまま **Save** をクリック
5. 数分後、生成されたURL（例: `https://yourname.github.io/repo-name/`）にアクセスすれば完了です。
